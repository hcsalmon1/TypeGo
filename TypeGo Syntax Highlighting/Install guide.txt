How to install TypeGo basic syntax highlighting:

Option 1 Command line:

-Open the command line
-Run command:
	code --install-extension typego-0.0.1.vsix
	
Option 2 VsCode UI:

-Open VsCode
-Click extensions
-Click ... button on the top right of the extensions window
-In that window, click: Install from VSIX...
-select the vsix file in this folder
